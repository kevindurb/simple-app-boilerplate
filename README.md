# Simple App Boilerplate

The shortest line to proof of concept, and a proof of concept for a future where
you may not need to transpile your javascript.

```
.
├── README.md
├── components
│   └── App.js
├── env
│   └── development.js
├── index.html
├── main.js
├── styles
│   ├── base.css
│   └── layout.css
└── vendor
    └── html.js
```
